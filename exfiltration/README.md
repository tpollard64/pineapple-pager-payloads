# Exfiltration

This folder contains payloads for extracting and exfiltrating data from target systems.

## Purpose
Exfiltration payloads are used to:
- Extract captured data from the Pineapple
- Transfer files and logs to remote servers
- Export network captures and collected information
- Automate data backup and retrieval

## Usage
Place your data exfiltration payloads in this directory. Ensure all payloads follow responsible disclosure practices and are used only in authorized testing scenarios.

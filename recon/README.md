# Recon

This folder contains reconnaissance payloads for gathering information about networks and targets.

## Purpose
Reconnaissance payloads are used to:
- Scan for wireless networks and devices
- Enumerate network services and hosts
- Gather information about connected clients
- Map network topology
- Identify potential targets

## Usage
Place your reconnaissance payloads in this directory. Each payload should document what information it collects and how it reports findings.

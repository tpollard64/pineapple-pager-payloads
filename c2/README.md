# Command and Control (C2)

This folder contains command and control payloads for managing the WiFi Pineapple Pager remotely.

## Purpose
C2 payloads are used to:
- Establish remote access to the Pineapple
- Enable remote command execution
- Maintain persistent connections
- Facilitate remote management and control

## Usage
Place your C2 payloads in this directory. All payloads should implement secure communication channels and be used only in authorized testing environments.

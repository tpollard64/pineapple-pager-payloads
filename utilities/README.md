# Utilities

This folder contains general utility payloads and helper scripts.

## Purpose
Utility payloads are used to:
- Automate common tasks
- Provide helper functions
- Manage Pineapple configuration
- Perform system maintenance
- Support other payload operations

## Usage
Place your utility scripts and helper payloads in this directory. These should be general-purpose tools that support various Pineapple operations.
